「情報の粒」がAIインフラを旅する — 全4製品 Three.js 3Dアニメーション

01
ARIES PCIe Retimer
SIGNAL RESTORATION
3Dメタファー PCB上のPCIeトレース（金色の線）を信号粒子が移動。左半分でノイズを纏いながら縮退し、チップパッケージ（中央）で完全再生→右半分で輝きながら到達。
実際の機能 PCIe差動ペア信号をボード中間でリタイミング（再整列・再増幅）。長距離配線でも信号品質をゼロ劣化に保つ。AIサーバーのGPU-CPU間接続に不可欠。
02
TAURUS Smart Cable Module
SCALE-OUT ETHERNET
3Dメタファー 5台のサーバーラック間を弧を描くケーブルが繋ぎ、中点に光る球＋回転リング（スマートノード）。データ粒子がケーブル上を流れ、ノード通過で増幅される。
実際の機能 EthernetでAIサーバー群を横展開するSCM。ケーブルモジュール内蔵ASICが信号品質・電力を自律最適化。大規模AIクラスタの必須インフラ。
03
SCORPIO Smart Fabric Switch
SCALE-UP AI FABRIC + HYPERCAST
3Dメタファー 中央のファブリックスイッチから8基のGPUへ放射状接続。通常ルートは1対1転送、Hypercastは中央で粒子が分裂し3つのGPUへ同時到達。
実際の機能 PCIeファブリックスイッチでGPU間を低遅延接続。Hypercast機能により、1回の送信で複数GPUへ同報配信。AIトレーニングの通信ボトルネックを解消。
04
LEO CXL Smart Memory Controller
CXL MEMORY EXPANSION
3Dメタファー CPU（左）からCXLバス経由でLeoコントローラー（中央）へ、さらに拡張メモリバンク（右）へ。DIMMスティックの積層構造とメモリ充填アニメーション。
実際の機能 CXLプロトコルでメモリプールを拡張。CPUのメモリ帯域制限を打破し、大規模AIモデルの学習・推論に必要な巨大メモリ空間を提供。
